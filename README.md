#Kalkulators
## Apraksts
šī programma atrisinās summu, atņems skaitļus, būs reizināšana, dalīšana un iznēst pakāpe. 
## Galvenās funkcijas
      -Ievada skaitļus
      -Atņems skaitļus
      -Summē skaitļus
      -Reizinā skaitļus
      -Dalā skaitļus
      -Reizinās pakāpe
## Izmantotās tehnoloģijas
      -C++
